# 🚀 Research AI Platform

An AI-powered research assistant that allows users to interact with scientific content using natural language.

## 🔧 Tech Stack
- FastAPI (Backend)
- HTML/CSS/JS (Frontend)
- Python
- Docker (optional)
- Azure (deployment-ready)

## ✨ Features
- 🔐 User authentication (JWT)
- 💬 AI chat interface
- ⚡ FastAPI backend APIs
- 📦 Scalable architecture (RAG-ready)

## 🚀 Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
